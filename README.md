# AI Career Copilot for Students

A website concept for an AI-powered career copilot that helps students apply to internships faster.

## Features

- Resume upload flow
- Job link analyzer
- ATS fit score
- Resume improvement suggestions
- Missing keyword detection
- Cover letter generation
- LinkedIn networking message generation
- Interview question prep
- International student support with CPT, OPT, and visa-friendly job signals
- Application tracker concept
